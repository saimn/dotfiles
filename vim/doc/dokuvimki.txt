INSTALLATION                                          *dokuvimki-installation*

Just unpack the tarball in your ~/.vim/ directory. Then edit your .vimrc and
add your settings (see CONFIGURATION). You'll also need to install the python
xmlrpclib module in order to use this plugin! For details about installing it
please refer to the documentation specific for your OS.

------------------------------------------------------------------------------
CONFIGURATION                                        *dokuvimki-configuration*

The following variables have to be present in your ~/.vimrc in order to get
DokuVimKi working.

g:DokuVimKi_USER            The username used to login into the remote wiki.

g:DokuVimKi_PASS            The password used to login into the remote wiki.

g:DokuVimKi_URL             The URL of the remote wiki (no trailing slash!).

g:DokuVimKi_TREEWIDTH       The width of the page tree window (default 35).

g:DokuVimKi_AUTOCONNECT     Set to "yes" to establish a XML-RPC connection to
                            your remote wiki whenever you start vim.

g:DokuVimKi_FOLDCOLWIDTH    Defines the width of the fold colums (default 3)
                            see :help foldcolumn for more information.

COMMANDS                                                  *dokuvimki-commands*

:DWEdit <page>              Opens the given wiki page in the edit buffer. If
                            the page does not exist on the remote wiki it will
                            be created once you issue :DWSend.

:DWSend <summary>           Sends the wiki page in the edit buffer to remote
                            wiki. If no edit summary is given it will be
                            saved as minor edit.

:DWBackLinks <page>         Loads a list of pages which link back to the given
                            wiki page into the edit buffer. If you are already
                            editing a page you can use the command without
                            supplying a wiki page and it will load a list of
                            backlinks to the page loaded in the edit buffer.

:DWList <expr>              Loads a list of wikipages where <expr> can be a
                            regular expression to limit the result. If no
                            expression is given a list of all available pages
                            on the remote wiki is shown. If you hit <Enter>
                            anywhere in this list, the page on the line the of
                            cursor will be loaded into the edit buffer.

:DWRevs <page> <offset>     Lists the available revisions of a wiki page. You 
                            can use an offset (integer) to view earlier 
                            revisions. The number of shown revisions depends 
                            on the $conf['recent'] setting of the remote wiki. 
                            By hitting <Enter> the revision under the cursor 
                            is loaded into the edit buffer.

:DWPageTree                 This will open a new window which displays a tree
                            like list of all available wiki pages of the
                            remote wiki. Like with :DWList, if you hit 
                            <Enter> the wiki page under the cursor will be
                            loaded into the edit buffer. Namespaces, for which
                            no wiki page exist are highlighted with a
                            preceeding '+' (those can be created like any
                            other wiki pages of course). DokuVimKi defines a
                            couple of mappings to ease the navigation in the
                            page tree see |dokuvimki-pageTree-mappings|.

:DWAuth                     Use this function if you want to re-authenticate
                            to a different remote wiki than specified in your 
                            .vimrc while editing. Simplechange the values of 
                            the configuration and issue :DWAuth.

EDIT-MAPPINGS                                        *dokuvimki-edit-mappings*

DokuVimKi mimics the functionality of DokuWikis edit toolbar to make it easier
to insert/edit wiki syntax. Unless otherwise noted all those mappings work
only in insert mode.

    <ALT+1>     insert Level1 headline

    <ALT+2>     insert Level2 headline

    <ALT+3>     insert Level3 headline

    <ALT+4>     insert Level4 headline

    <ALT+5>     insert Level5 headline

    <ALT+B>     insert bold text

    <ALT+I>     insert italic text

    <ALT+U>     insert underlined text

    <ALT+C>     insert monospaced text

    <ALT+D>     insert deleted text

    <ALT+L>     insert wiki link

    <ALT+E>     opens a link under the current cursor position for editing
                (this mappings only works in normal mode)

------------------------------------------------------------------------------
PAGETREE-MAPPINGS                                *dokuvimki-pageTree-mappings*

DokuVimKi comes with a couple of mappings to easy the usage of the page tree
window.

    +           unfolds the fold under the cursor

    -           folds the fold under the cursor

    *           unfolds all folds
 
    _           folds all folds
